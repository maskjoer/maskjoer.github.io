文章录入模板

复制下面对象到 posts.js 的数组最前面，再替换为你的内容。

{
  slug: "your-unique-slug",
  title: "你的文章标题",
  date: "2026.03.17",
  category: "前端",
  readingTime: "6 分钟",
  tags: ["博客", "实战", "总结"],
  summary: "这篇文章的一句话摘要。",
  content: [
    "第一段正文。",
    "第二段正文。",
    "第三段正文。"
  ]
}

发布命令

git add posts.js
git commit -m "feat: add new post"
git push origin main
